# Bitcoin-LoRa-Mesh
A decentralized Bitcoin Mesh network that communicates over long-range LoRa radio, enabling off-grid transactions
<br>
Bitcoin Becomes Truly Uncensorable<br>
Broadcast Bitcoin transactions over LoRa mesh networks<br>
No Internet required at the sender's location<br>
<br>
## The nRF52840 E22P868M30S Solar LoRa Node
<br>
Ultra Low Power Solar LoRa Node up to 30 dBm with 8000mAh Li-ion <br>
Can operate for weeks during the dark days of winter<br>
Compatible with Reticulum, Meshcore, Meshtastic, ...<br>
